### MyCurrency
Pet-проект с API для сравнения курсов валют

---
Для запуска проекта в терминале из папки client команда: **npm run dev**
<br/>
Проект будет запущен на порту localhost:3000

<br/>

---
### Stack

- JavaScript
- React
- Redux-toolkit
- Bootstrap, React-Bootstrap
- Material UI
- Axios
- uuid
